# Swiss Tax Calculator

A frontend-only tax calculator for Swiss residents. Calculates federal, cantonal, and municipal taxes based on income, deductions, and personal situation.

## Features

- Progressive tax bracket calculations for federal and cantonal taxes
- Support for single and married tariffs
- Municipal tax multipliers
- Church tax calculations by religion
- Pillar 3a deduction handling
- Compare tax burden across municipalities

## Tech Stack

React, TypeScript, Vite, Tailwind CSS, Recharts

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Available Scripts

```bash
npm run dev      # Start development server
```
